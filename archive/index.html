Dublin Bus Realtime

How to use

  • Dublin Bus Realtime lets you
  • Enter the bus stop ID into the search bar and hit enter. The bus stop ID - or stop no. -  is shown on the bus stop pole. Searching the stop ID will return the live departure times from that bus stop.
  • Alternatively, you can search for an address and get the closest bus stops. Example: searching for "Talbot St" will retrieve all bus stops in a radius of 500 meter. Clicking on the bus stop will fetch the real time departures for that bus stop.
  • Click on a bus line number to get the stops along the way. Clicking on one of the stops will load the real time data for that stop.

Search by address to find nearby bus stops. For example: "1, Howth Road" or "Talbot St"

What's new?

  • New material cards interface, loading animations when fetching live data from API.
  • Bus chips for every stop, upon clicked, all bus stops for that route will be shown. Clicking on another bus stop will redirect to live data for that particular bus stop.
  • Autocomplete functionality for searching bus stops (currently only triggered for numeric values).
  • Maps link for the relevant bus stop.
  • Mobile improvements.
